# AI Video Editor (MVP)

## Goal
Build a web-based AI video editing app that provides niche-based short video templates and saves creators hours of editing.

## MVP Features
- Select niche
- View templates
- Upload video
- Apply template
- Export video

## Tech Stack
Frontend: React + Tailwind
Backend: FastAPI
Database: MongoDB
Video Processing: FFmpeg

## Rules
- MVP only
- No heavy AI
- Deadline: Friday
